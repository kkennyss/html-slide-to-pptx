Features

🚀 Zero Setup

No Python, no CLI, no dependencies. Open the HTML and click Download.

How It Works

Primary Path

HTML slides → dom-to-pptx (CDN JS library) → Native PPTX shapes & text boxes → Editable in PowerPoint

Fallback Path

HTML slides → Headless Chrome screenshots → python-pptx assembly → Image-based PPTX

Design Principles

Feel over form — a slide is "饱满" (full) when its content is vertically substantial, regardless of layout type.

Padding compensates for content volume: heavy content needs ~30px, light content needs ~60px.

When a slide feels thin, add context instead of forcing format changes.
